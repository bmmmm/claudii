Install

$ brew tap bmmmm/tap && brew install claudii
$ claudii on   # enable the in-session status line

Three layers, all read-only

  • 1. CC-Statusline — dense metrics inside Claude Code, every turn
  • 2. Session Dashboard — active sessions above your shell prompt
  • 3. ClaudeStatus — model health in your RPROMPT, never blocks

Cost, token & cache data read straight from Claude Code's local session JSONL. No telemetry, no API calls on your behalf — nothing leaves your machine.
